HEAD TO OUTLOOK

Microsoft Outlook Microinteractions: Beta

TRIGGERS

Trash Button

  • Hover ON
  • Hover OFF
  • Mouse DOWN
  • Mouse UP

RULES

Trash Button

  • Hover ON
    • Cursor changes to finger
    • Icon changes color to black
    • 3-second delay "Delete" appears as tagline
  • Hover OFF
    • Cursor changes back to pointer
    • "Delete" tagline disappears
  • Mouse UP
    • Popup with "Undo" button appears, fades after 5 seconds
    • Trash count increases by 1

FEEDBACK

Trash Button

  • Hover ON
    • Cursor changes to finger
    • Icon color changes to black
    • Displays "Delete" after 3 seconds
  • Mouse UP
    • Popup with "Undo" button appears
    • Popup fades away after 5 seconds

LOOPS & MODES

Trash Button

  • No loops
  • Popup mode with "Undo" button appears upon delete
