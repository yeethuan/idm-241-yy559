RULES

Checkbox - DEFAULT

  • Hover ON
    • Cursor changes to finger
    • Gray outline turns a dark gray
  • Hover OFF
    • Finger changes to cursor
    • Gray outline resumes regular gray
  • Mouse DOWN
    • Gray outline turns to a light gray
  • Mouse UP
    • Checkbox background fills in dark blue
    • Checkbox outline turns matching dark blue
    • White checkmark appears inside the checkbox
    • Email is selected for further action

Checkbox - ACTIVE

  • Hover ON
    • Cursor changes to finger
    • Checkbox background turns a dark blue
    • Checkbox outline turns a matching dark blue
  • Hover OFF
    • Finger changes to cursor
    • Checkbox background resumes regular blue
    • Checkbox outline turns a matching regular blue
  • Mouse DOWN
    • Checkbox background turns light blue
    • Checkbox outline turns matching light blue
  • Mouse UP
    • Checkbox background turns white
    • Checkbox outline turns regular gray
    • Checkmark disappears
    • Email is no longer selected for further action

Star Button - DEFAULT

  • Hover ON
    • Cursor changes to finger
  • Hover OFF
    • Finger changes to cursor
  • Mouse DOWN
    • Star is filled in yellow
    • Star outline turns matching yellow
  • Mouse UP
    • Star animates bursting effect releasing matching yellow particles lasting for 1s

Star Button - ACTIVE

  • Hover ON
    • Cursor changes to finger
  • Hover OFF
    • Finger changes to cursor
  • Mouse UP
    • Star is filled in white
    • Star outline turns black

Pin Button - DEFAULT

  • Hover ON
    • Cursor changes to finger
    • Icon .3s animation ease into blue
    • 3s delay “Pin this email” appears as tagline
  • Hover OFF
    • Finger changes to cursor
    • Icon .3s animation ease into gray
    • “Pin this email” disappears as tagline
  • Mouse DOWN
    • Icon shrinks .9
  • Mouse UP
    • Icon unshrinks
    • Email-container turns light blue
    • Email-container outline turns dark blue
    • Icon jumps
    • Icon remains blue for ACTIVE state
    • Database record set to TRUE
    • Email-container is moved to the top of the Emails

Pin Button - ACTIVE

  • Hover ON
    • Cursor changes to finger
    • 3s delay “Unpin this email” appears as tagline
  • Hover OFF
    • Finger changes to cursor
    • “Unpin this email” disappears as tagline
  • Mouse DOWN
    • Icon shrinks .9
  • Mouse UP
    • Icon unshrinks
    • Email-container returns color white
    • Email-container outline returns to gray
    • Icon returns to color gray
    • Database record set to FALSE
    • Email-container is returned to its original location

Response Button

  • Hover ON
    • Cursor changes to finger
    • Icon .3s animation ease into blue
    • 3s delay Respond to email” appears as tagline
  • Hover OFF
    • Finger changes to cursor
    • Icon .3s animation ease into gray
    • 3s delay “Respond to email” disappears as tagline
  • Mouse DOWN
    • Icon shrinks .9
  • Mouse UP
    • Modal of input boxes appears to respond to email which includes respond and cancel button that close out the modal

Trash Button

  • Hover ON
    • Cursor changes to finger
    • Icon .3s animation ease into blue
    • 3s delay "Delete this email” appears as tagline
  • Hover OFF
    • Finger changes to cursor
    • Icon .3s animation ease into gray
    • 3s delay “Delete this email” disappears as tagline
  • Mouse DOWN
    • Icon shrinks .9
  • Mouse UP
    • Email slides to the right and disappears
    • Database record set to FALSE
    • Trash number set to n+1
    • Trash database set to n+1
    • A popup appears at the bottom of the page confirming the email has been deleted, with an “Undo” button that causes the deleted email to reappear when triggered
    • Popup fades away after 5s of inaction
