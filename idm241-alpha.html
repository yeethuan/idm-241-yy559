Alpha Description

Microsoft Outlook: Trash

HEAD TO OUTLOOK

TRIGGERS

Trash Button

  • Hover ON
  • Hover OFF
  • Mouse DOWN
  • Mouse UP

RULES

Trash Button

  • Hover ON
    • Cursor changes to finger
    • Trash lid animates shake
    • Icon background fills in with the color red 20%
    • Icon outline changes to the color red
    • 3 second delay "Delete" appears as tagline
  • Hover OFF
    • Cursor changes back to pointer
  • Mouse UP
    • Database record flag set to FALSE

FEEDBACK

Trash Button

  • Hover ON
    • Cursor changes to finger
    • Trash lid animates shake
    • Icon background fills in with the color red 20%
    • Icon outline changes to the color red
    • 3 second delay "Delete" appears as tagline
  • Hover OFF
    • Cursor changes back to pointer

LOOPS & MODES

Trash Button LOOPS

  • NONE

Trash Button MODES

  • A popup appears at the bottom of the page confirming the message has been deleted, with an "Undo" button allowing the user to reverse the action.
